# Stupid System Data — Guide

This repository provides a small Next.js template and a written guide for inspecting and clearing "System Data" on macOS. The website for this is [clearingsystemdata](https://chuahchenghang.github.io/clearingsystemdata/)

## What & Why

This project was built to be two things at once: a practical how-to for freeing up space used by System Data on a Mac, and a minimal, editable Next.js article template. You can use it to publish a short guide (what to do and why), and you can edit the content directly in code so it's easy to script, version, or reuse as a template for other articles.
